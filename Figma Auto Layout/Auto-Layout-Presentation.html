What is Auto Layout?

The foundation of responsive design in Figma

🎯

Smart Containers

Auto Layout transforms static frames into intelligent containers that automatically resize and reposition content based on predefined rules.

⚡

Like CSS Flexbox

If you're familiar with web development, Auto Layout works similarly to CSS Flexbox, giving you powerful layout control with minimal effort.

🔄

Dynamic Adaptation

Content changes? No problem. Auto Layout automatically adjusts spacing, sizing, and positioning without manual tweaking.

🚀

Design Faster

Build responsive components and design systems 10x faster. Update once, propagate everywhere. Perfect for maintaining consistency.

3 Ways to Activate Auto Layout

Choose the method that fits your workflow

⌨️

Keyboard Shortcut

The fastest way for power users

  • Select frame, group, or objects
  • Press Shift + A
  • Instantly converts to Auto Layout
📐

Right Sidebar

Visual approach using the Design panel

  • Select your frame or objects
  • Find "Auto Layout" section in sidebar
  • Click the + button
🖱️

Top Toolbar

Great for beginners learning the interface

  • Select your objects
  • Click + icon in top toolbar
  • Select "Add Auto Layout"
💜

Visual Indicator

How to know it's working

  • Purple outline appears around frame
  • Purple resize handles show up
  • Properties panel updates with options

Key Auto Layout Properties

Master these controls to unlock full potential

Direction
Controls how child elements stack: Horizontal (left to right) or Vertical (top to bottom). Essential for layout structure.
Spacing
Gap between child elements. Use 8pt multiples (8px, 16px, 24px) for consistent design system spacing.
Padding
Space inside the frame around content. Can be set uniformly or individually for top, right, bottom, left.
Alignment
How items align within container: start, center, end, or space-between. Critical for visual balance.
Resizing - Hug
Frame automatically wraps tightly around its content. Perfect for buttons and dynamic components.
Resizing - Fill
Child elements expand to fill available space in container. Great for responsive designs and flexible layouts.

Demo Case 1: Responsive Button

Create buttons that adapt to text length automatically

🎯 Use Case

Build a button that automatically adjusts its width based on text content while maintaining consistent padding.

📋 Steps to Create

  1. Create a text layer with "Click Me"
  2. Select the text and press Shift + A
  3. Set padding: 16px horizontal, 12px vertical
  4. Set alignment to Center
  5. Apply button background color and corner radius (8px)

✨ Result

Change text from "Click Me" to "Submit Application" → Button automatically expands! Maintains perfect padding and proportions without any manual adjustment.

💡 Why It's Useful

  • Maintains design consistency across all button variations
  • Speeds up creation of Primary, Secondary, Tertiary buttons
  • Easy to update text without breaking layout
  • Perfect for design systems and component libraries

Demo Case 2: Navigation Bar

Build horizontal menus with perfect spacing

🎯 Use Case

Create a horizontal navigation menu where items are evenly spaced and easy to add or remove.

📋 Steps to Create

  1. Create text layers: Home, About, Services, Contact
  2. Select all text layers and press Shift + A
  3. Set direction to Horizontal
  4. Set spacing between items: 32px
  5. Add padding all around: 20px
  6. Set alignment to Center

✨ Result

A perfectly spaced nav bar that automatically adjusts when you add or remove menu items. No manual repositioning needed!

💡 Why It's Useful

  • Consistent spacing without manual measurements
  • Instantly add/remove navigation items
  • Quick to create mobile, tablet, desktop variations

Demo Case 3: Card Component

Design system cards that adapt to content

🎯 Use Case

Design a content card with image, title, description, and button that maintains structure regardless of content length.

📋 Steps to Create

  1. Create: Image placeholder, Title text, Description, Button
  2. Select all elements and press Shift + A
  3. Set direction to Vertical
  4. Set spacing: 16px between elements
  5. Add padding: 24px all around
  6. Set fixed width: 320px

✨ Result

Card height automatically adjusts based on description length, image size, or added elements. Perfect vertical rhythm maintained!

💡 Why It's Useful

  • Essential for design systems and component libraries
  • Cards adapt to different content automatically
  • Maintains consistent vertical spacing and rhythm

Demo Case 4: Form with Fields

Create perfectly structured forms in minutes

🎯 Use Case

Build a form where each field (label + input) is consistently spaced and easy to maintain.

📋 Steps - Individual Field

  1. Create label text and input rectangle
  2. Select both, press Shift + A
  3. Direction: Vertical
  4. Spacing: 8px
  5. Alignment: Left

📋 Steps - Full Form

  1. Create field pairs: Name, Email, Password
  2. Select all groups, press Shift + A
  3. Direction: Vertical
  4. Spacing: 24px between fields
  5. Padding: 32px

✨ Result

Perfect form structure! Add or remove fields, everything stays aligned and properly spaced.

💡 Why It's Useful

  • Rapid form prototyping for any project
  • Consistent field spacing across all forms
  • Easy to maintain and update field structure

Demo Case 5: Dynamic List Items

Perfect for to-do lists, messages, and menus

🎯 Use Case

Create a list (task list, chat thread, menu items) where items can be easily added or removed.

📋 Steps - Single Item

  1. Create: Icon/checkbox, Text, Delete button
  2. Select all parts, press Shift + A
  3. Direction: Horizontal
  4. Spacing: 12px
  5. Padding: 16px
  6. Alignment: Center vertically

📋 Steps - Full List

  1. Duplicate item 3-4 times
  2. Select all items, press Shift + A
  3. Direction: Vertical
  4. Spacing: 8px

✨ Result

Add items by duplicating → automatic stacking! Delete items → list adjusts perfectly. No manual work!

💡 Why It's Useful

  • Perfect for to-do lists, chat interfaces, menu items
  • Instant updates when content changes
  • Professional, consistent appearance automatically

Pro Tips & Advanced Techniques

Level up your Auto Layout mastery

🎭 Nested Auto Layouts
Nest Auto Layout frames inside each other for complex, responsive designs. Example: Card with Auto Layout → Contains header with Auto Layout → Contains icon + title with Auto Layout.
🤗 Hug Contents
Set frame resizing to "Hug contents" to make the frame automatically match its content size. Perfect for buttons, badges, and tags that need to wrap tightly around text.
📏 Fill Container
Set child elements to "Fill container" to make them expand to available space. Great for creating responsive layouts where elements should stretch to fill their parent.
📍 Absolute Position
Hold Alt (Win) or Option (Mac) while placing an element to keep it in absolute position. Won't follow Auto Layout rules - useful for overlays and floating elements.
⌨️ Keyboard Shortcuts
• Shift + A - Add Auto Layout
• Alt + Arrow Keys - Adjust spacing between items
• Shift + Arrow Keys - Adjust padding
🎨 8-Point Grid System
Always use multiples of 8 for spacing and padding (8px, 16px, 24px, 32px). This creates visual consistency and makes your designs feel more polished and professional.
