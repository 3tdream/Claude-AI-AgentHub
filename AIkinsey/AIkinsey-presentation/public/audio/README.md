# Audio Files

This folder contains background music and sound effects for the presentation.

## Adding Audio Files

1. **Background Music**: Place your background music file here as:
   - `background-music.mp3` (primary format)
   - `background-music.ogg` (fallback format)

2. **Supported Formats**:
   - MP3 (recommended for compatibility)
   - OGG (fallback option)
   - WAV (high quality, larger file size)

3. **File Size Recommendations**:
   - Keep files under 5MB for fast loading
   - Use 128kbps or higher bitrate for good quality
   - Loop-friendly tracks work best (seamless loop points)

## Current Audio Files

Place your audio files in this directory and they will be automatically detected by the presentation.

## Testing Audio

1. Start the development server: `npm run dev`
2. Open the presentation in your browser
3. Click the sound icon in the top navigation
4. Audio should start playing if files are present
