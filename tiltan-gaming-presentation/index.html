Lesson 2: Market & Players 17 / 30

Core Loop מול Meta

  • Core Loop = פעולה חוזרת עכשיו (ספין / פרס / בנייה)
  • Meta = התקדמות לאורך זמן (קלפים, כפרים, אירועים)
  • Core = משחק רגעי / Meta = משחק ממכר
Core Loop עושה כיף. Meta עושה התמכרות.
