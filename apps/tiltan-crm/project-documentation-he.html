מה זה Tiltan CRM?

Tiltan CRM היא מערכת ניהול מקיפה שנבנתה במיוחד עבור מכללת טילטן לעיצוב ותקשורת חזותית בחיפה. המערכת מאפשרת ניהול יעיל של כל המידע הקשור לסטודנטים, הרצאות ומרצים במקום אחד.

המערכת פותחה עם טכנולוגיות מודרניות ומציעה ממשק משתמש נוח וקל לשימוש, המאפשר לצוות המכללה לנהל את הפעילות היומיומית בצורה פשוטה ויעילה.

טכנולוגיות בשימוש

Frontend (ממשק משתמש)

  • Next.js 14 - פריימוורק React מתקדם
  • TypeScript - שפת תכנות מודרנית ובטוחה
  • Tailwind CSS - עיצוב מודרני ורספונסיבי
  • shadcn/ui - ספריית קומפוננטות עיצוב
  • Lucide Icons - אייקונים מעוצבים

Backend (שרת ומסד נתונים)

  • Next.js API Routes - שרת API מובנה
  • LowDB - מסד נתונים מבוסס JSON
  • Node.js - סביבת הרצה
